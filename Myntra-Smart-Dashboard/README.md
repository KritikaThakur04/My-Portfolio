
# Myntra Smart Dashboard

## Project Overview

This project is an interactive dashboard developed using **R Shiny** to analyze Myntra product data. The dashboard allows users to explore products based on category, brand, price range, and customer ratings.

## Objective

The objective of this project was to analyze product data and build an interactive dashboard that helps users easily understand product distribution and compare brands.

## Tools & Technologies

* R Programming
* R Shiny
* Tidyverse
* ggplot2
* dplyr

## Dataset

* Myntra Product Dataset
* More than 168,000 product records

## Features

* Filter products by category
* Filter products by price range
* Filter products by ratings
* Brand-wise product analysis
* Interactive charts and KPI cards

## Data Cleaning

Before analysis, the dataset was cleaned by:

* Handling missing values
* Removing duplicate records
* Checking data types
* Organizing data for analysis

## Key Insights

* Identified brands with the highest number of products.
* Compared products across different price ranges.
* Analyzed customer ratings across brands and categories.
* Built an interactive dashboard for easier data exploration.
