# Excel Dashboard Project

## Objective
I created a Global Development Dashboard using World Bank and UNDP (HDI) data to visualize long-term trends in GDP per capita, life expectancy, HDI, and internet usage across 217 countries.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## Features
- Interactive dashboard
- Pivot tables for data summarization
- Charts for visualization
- Slicers for filtering data

## Skills Learned
- Data analysis
- Dashboard creation
- Data visualization
- Reporting
